SELECT 문
테이블내에 저장되어있는 데이터를 조회하기 위해 사용하는 명령어
1) 기본 형식
select 컬럼명1, 컬럼명2 as 별명 from 테이블명
where 조건
order by 컬럼명
limit 숫자

2) where 절
: 조건절. 원하는 데이터를 가져올 때 사용

3) order by
데이터를 순서대로 정렬할 때 사용
기본적으로 오름차순 정렬. 내림차순 정렬시 desc를 사용

4) limit
조회하는 데이터의 개수를 지정할 때 사용. 
order by와 연결하여 상위 몇개 또는 하위 몇개를 조회할때 주로 사용

5) distinct
데이터를 조회할때 중복 데이터를 빼고 한번만 보이고자 사용
주의, 단순히 데이터의 중복을 빼는 역할이며 그룹과는 다른 개념!!


<연산자>
1) 산술연산자
+ - * / %
2) 비교연산자
>=, <=, >, <, =, !=, <>
같지않다는 의미의 <> 기억 필요
3) 논리연산자
AND, OR, NOT
4) 집합연산자
UNION 
: 여러 테이블을 하나로 합칠때 사용하나, 별로 사용하지 않음
5) IS NULL, IS NOT NULL
: 알 수 없는 값(Unknown Value)의 의미. '' 빈 문자열과는 다른 개념
6) IN
: OR연산자를 수행해야하는 경우 좀 더 간단하게 사용할 수 있음
7) BETWEEN AND
: ~이상, ~이하라는 조건을 수행할 수 있음
8) LIKE
: 문자열의 패턴을 지정할 수 있음
% 0개의 문자를 가진 문자열을 포함한다는 의미
_  정확히 한개의 문자를 포함한다는 의미
